Detailed Description

template<class ValT, gclp_char CharT = char, class Traits = std::char_traits<CharT>>
class gclp::basic_optional< ValT, CharT, Traits >

Represents an optional command-line parameter.

This class extends basic_cl_param to handle optional command-line parameters with specific key characters, key strings, and a brief description.

Template Parameters
ValTThe type of the parameter value.
CharTThe character type used for keys (default is char).
TraitsThe character traits type (default is std::char_traits<CharT>).

inlinenoexcept

Provides access to the default value adaptor for setting default values of the parameter.

This member function allows direct access to the default value adaptor associated with the parameter. It enables setting default values for the parameter using the defval member functions provided by the adaptor.

Example Usage:

// Sets the default value of the parameter to 42 using the default value adaptor.
auto param = gclp::optional<int>{'i', "integer", "An optional integer parameter"}->defval(42);
