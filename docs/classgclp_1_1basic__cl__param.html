Detailed Description

template<class ValT, gclp_char CharT = char, class Traits = std::char_traits<CharT>>
class gclp::basic_cl_param< ValT, CharT, Traits >

Represents a basic command-line parameter with specified key characters and brief description.

This class provides functionality to handle command-line parameters with specific key characters and brief descriptions.

Template Parameters
ValTThe type of the parameter value.
CharTThe character type used for keys (default is char).
TraitsThe character traits type (default is std::char_traits<CharT>).
